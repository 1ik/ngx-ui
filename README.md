# style-guide
Style guide for the Swimlane platform

## Frameworks
- [sass](http://sass-lang.com/)
- [suitcss](https://suitcss.github.io/)

## Run
- `npm i`
- `npm start`
- Browse to [http://localhost:999](http://localhost:9999)
